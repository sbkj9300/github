## わかったことまとめ
- 変数を格納する領域は2つある。 
    - スタック領域とヒープ領域
- スタック領域にはprimitive型が格納される。
    - intとかは値がそのまま、Stringとかはヒープ領域のどのメモリに格納してあるかの場所が格納される。(上の1日目参照)
- どこに格納されている値を見ているかを意識することが大事
